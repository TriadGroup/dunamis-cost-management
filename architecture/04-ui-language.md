# 04 — UI Language

## Linguagem padrão
- usar linguagem do campo
- evitar jargão corporativo
- evitar inglês desnecessário

## Traduções travadas
- `markup` → `acréscimo`
- `allocation` → `rateio` ou `valor lançado`
- `appropriated` → `valor apropriado`
- `inventory lot` → `lote do estoque`
- `marketable` → `vendável`
- `usage` → `uso`
- `application` → `aplicação`

## Formatação
- dinheiro sempre com `R$`
- margem e perda sempre com `%`
- custo sempre com unidade explícita

## Ajuda contextual
Termos sensíveis devem poder receber `!` com explicação curta.

Exemplos:
- `Vendável`: o que sobrou bom para vender
- `Valor apropriado`: o valor que realmente entrou na conta desta cultura
- `Residual`: o efeito que continua no solo ou na área nos próximos ciclos
